# Josef Menezes CV
mywebsite:https://josefmenezes.github.io/cvbyj/

## Overview
This project is a simple HTML-based Curriculum Vitae (CV) for Josef Menezes, a Data Scientist. It showcases personal information, teaching courses, work experience, skills, and links to additional pages like hobbies and contact details.

## Installation
No installation is required. Ensure you have a web browser installed on your system.

## Usage
1. Download or clone the project files.
2. Open the `imagescv.html` file in any modern web browser (e.g., Chrome, Firefox, Safari).
3. Navigate through the sections to view the CV content.

# Images Directory

This directory contains image assets and related files for the CV project of Josef Menezes.

## Contents

- `profile.jpg`: A high-resolution profile picture used for personal branding and professional presentation in the CV. Reason: A profile image adds a personal touch, making the CV more engaging and memorable for viewers, which is essential in web-based portfolios to humanize the candidate.

- `imagescv.html`: An HTML file that displays the CV with embedded images. Reason: This file integrates images directly into the CV layout, allowing for a visually rich web page that showcases the profile picture and any other visuals, improving user experience and accessibility compared to plain text or PDF formats.

## Usage

These assets are optimized for web use to ensure fast loading times. The HTML file serves as a standalone page for viewing the CV online. Ensure all images are in JPEG format for broad compatibility and compression to reduce file sizes without significant quality loss.
